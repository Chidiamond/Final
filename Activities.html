WEB DESIGN

Website is a place connected to the intenet where people advertise or create things on the internet. web designers build web pages using HTML tags that defines content of each page.The appearance and element within a web page are typically defined using CSS.Therefore most website include a combination of HTML&CSS that defines how each page will apper in a browser.

IMPORTANCE OF A WEBSITE

1.Well looked professional website is the door to online market and a base to build your clientele. 2.Low cost advertising:The advert posted on your website is not as expensive as that of billboards. 3.Sales:It boost sales since potential customers are not restricted to business. 4.Accessibility:Your company will be easily accessible and located. 5.Good web design enhances effectiveness such that it delivers the right message into the heart of customers.

BENEFITS OF WEBSITE

It keeps people up-to-date with news and information about your company. It increases the number of people who finds your web site through search engines. It is fast and easy to use
